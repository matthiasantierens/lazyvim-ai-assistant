# Roadmap

Future development ideas for lazyvim-ai-assistant. No timeline or commitment.

For v2.0.0 release notes, see [CHANGELOG.md](CHANGELOG.md).

---

## Future Considerations

### MCP (Model Context Protocol) Support
- Integration with external tool servers via MCP
- Project-specific tool configurations
- Custom tools defined by MCP servers

### Multiple Provider Support
- Direct Ollama integration (bypass LM Studio)
- OpenAI API support
- Anthropic API support
- Provider switching without restart

### Enhanced Context Awareness
- Semantic code search (embeddings-based)
- Cross-file dependency graph
- Test coverage context
- LSP diagnostics integration

### Workflow Automation
- Multi-step predefined workflows
- Chain prompts together
- Automated code review pipelines
